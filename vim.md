
# Vim 常用命令总结表

## 1. 模式切换

| 类别     | 命令        | 作用说明                     |
|----------|-------------|------------------------------|
| 回普通模式 | `Esc`      | 从插入 / 可视等模式回到普通模式 |
| 插入模式 | `i`         | 在光标前插入                 |
| 插入模式 | `a`         | 在光标后插入                 |
| 插入模式 | `o`         | 在当前行下方新开一行并插入   |
| 插入模式 | `O`         | 在当前行上方新开一行并插入   |
| 插入模式 | `I`         | 在行首插入                   |
| 插入模式 | `A`         | 在行尾插入                   |
| 可视模式 | `v`         | 字符可视模式（选中字符）     |
| 可视模式 | `V`         | 行可视模式（选中整行）       |
| 可视模式 | `Ctrl+v`    | 块可视模式（矩形选中）       |

## 2. 光标移动

| 类别     | 命令        | 作用说明                          |
|----------|-------------|-----------------------------------|
| 基本移动 | `h`         | 左移一个字符                      |
| 基本移动 | `j`         | 下移一行                          |
| 基本移动 | `k`         | 上移一行                          |
| 基本移动 | `l`         | 右移一个字符                      |
| 行内移动 | `0`         | 移动到行首                        |
| 行内移动 | `$`         | 移动到行尾                        |
| 单词移动 | `w`         | 跳到下一个单词开头                |
| 单词移动 | `b`         | 跳到上一个单词开头                |
| 文件移动 | `gg`        | 跳到文件开头                      |
| 文件移动 | `G`         | 跳到文件末尾                      |
| 跳到行   | `:n`        | 跳到第 n 行（如 `:10`）           |

## 3. 保存 / 退出 / 撤销

| 类别     | 命令         | 作用说明                      |
|----------|--------------|-------------------------------|
| 保存     | `:w`         | 保存当前文件                  |
| 退出     | `:q`         | 退出（未保存会阻止）          |
| 保存退出 | `:wq`        | 保存并退出                    |
| 强制退出 | `:q!`        | 不保存强制退出                |
| 另存为   | `:w filename`| 另存为指定文件                |
| 撤销     | `u`          | 撤销上一步操作                |
| 重做     | `Ctrl+r`     | 重做被撤销的操作              |

## 4. 删除 / 复制 / 粘贴

| 类别     | 命令        | 作用说明                          |
|----------|-------------|-----------------------------------|
| 删除字符 | `x`         | 删除光标所在字符                  |
| 删除单词 | `dw`        | 删除从光标到单词结尾              |
| 删除行   | `dd`        | 删除当前行                        |
| 删除多行 | `n dd`      | 删除从当前行开始的 n 行，如 `3dd` |
| 删到行尾 | `d$`        | 删除从光标到行尾                  |
| 删到行首 | `d0`        | 删除从光标到行首                  |
| 复制行   | `yy`        | 复制当前行                        |
| 复制多行 | `n yy`      | 复制从当前行开始的 n 行           |
| 复制到尾 | `y$`        | 复制从光标到行尾                  |
| 粘贴后   | `p`         | 在光标后（下一行/后面）粘贴      |
| 粘贴前   | `P`         | 在光标前（上一行/前面）粘贴      |

## 5. 查找与替换

| 类别       | 命令                 | 作用说明                                      |
|------------|----------------------|-----------------------------------------------|
| 向下查找   | `/text`              | 从当前光标向下查找 `text`                     |
| 向上查找   | `?text`              | 从当前光标向上查找 `text`                     |
| 下一个匹配 | `n`                  | 跳到下一个匹配                                |
| 上一个匹配 | `N`                  | 跳到上一个匹配                                |
| 当前行替换 | `:s/old/new/`        | 当前行第一个 `old` → `new`                   |
| 当前行全替换 | `:s/old/new/g`     | 当前行所有 `old` → `new`                     |
| 全文替换   | `:%s/old/new/g`      | 全文件所有 `old` → `new`                     |
| 交互替换   | `:%s/old/new/gc`     | 全文替换，逐个确认（`c` 表示 confirm）       |

## 6. 缩进与对齐

| 类别     | 命令         | 作用说明                          |
|----------|--------------|-----------------------------------|
| 行缩进   | `>>`         | 当前行右缩进                      |
| 行缩进   | `<<`         | 当前行左缩进                      |
| 多行缩进 | `选中 + >`   | 可视模式选中后右缩进              |
| 多行反缩 | `选中 + <`   | 可视模式选中后左缩进              |
| 自动缩进 | `=`          | 对选中区域做自动缩进              |

## 7. 可视模式常用操作

| 类别         | 命令         | 作用说明                          |
|--------------|--------------|-----------------------------------|
| 进入字符选中 | `v`          | 开始按字符选中                    |
| 进入行选中   | `V`          | 开始按行选中                      |
| 进入块选中   | `Ctrl+v`     | 开始块/矩形选中                   |
| 复制选区     | `y`          | 在可视模式下复制选区              |
| 删除选区     | `d`          | 在可视模式下删除选区              |
| 缩进选区     | `>` / `<`    | 在可视模式下缩进 / 反缩进         |

## 8. 分屏与标签页

| 类别       | 命令             | 作用说明                           |
|------------|------------------|------------------------------------|
| 水平分屏   | `:sp`            | 水平分割窗口                       |
| 垂直分屏   | `:vsp`           | 垂直分割窗口                       |
| 分屏切换   | `Ctrl+w h/j/k/l` | 在分屏之间移动光标                 |
| 新标签页   | `:tabe file`     | 在新标签页中打开文件              |
| 下一个标签 | `gt`             | 切换到下一个标签页                 |
| 上一个标签 | `gT`             | 切换到上一个标签页                 |

## 9. 其他高频小技巧

| 类别         | 命令    | 作用说明                     |
|--------------|---------|------------------------------|
| 重复上次操作 | `.`     | 重复上一次“修改”操作        |
| 取消高亮     | `:noh`  | 取消搜索高亮结果             |
